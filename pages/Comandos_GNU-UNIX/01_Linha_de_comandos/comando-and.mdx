# AND lógico
O operador lógico "AND" é representado pelo símbolo && e exige que o primeiro comando seja executado com sucesso para que o segundo seja executado. 

Aqui está um exemplo:
`ls /proc && cat /etc/passwd && cat /etc/group`

Neste caso, os dois comandos são executados.

<CH.Spotlight>

```bash terminal.sh
ls /proc && cat /etc/passwd
```

---

Vamos listar a saida de dois comandos usando &&

```bash terminal.sh
ls /proc && cat /etc/passwd
```

---

Lista conteúdo da pasta /proc

```bash terminal.sh focus=1[1:8]
ls /proc && cat /etc/passwd
```

---

Uso operador lógico &&

```bash terminal.sh focus=1[10:11]
ls /proc && cat /etc/passwd
```

---

Adicionado comando cat mostrando passwd

```bash terminal.sh focus=1[13:28]
ls /proc && cat /etc/passwd
```
